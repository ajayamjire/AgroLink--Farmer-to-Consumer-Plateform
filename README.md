# AgroLink 🌾

AgroLink is a smart digital marketplace platform that connects farmers directly with consumers, retailers, and local vendors. The platform removes middlemen from the agricultural supply chain and helps farmers sell their products at fair prices while consumers get fresh farm produce at affordable rates.

---

## 👨‍🌾 Farmer Features

* Farmer Registration & Login
* Add Product Listings
* Upload Product Images
* Manage Product Price & Quantity

* User Registration & Login

# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/yourusername/agrolink.git
```

## 2️⃣ Open Project Folder

```bash
cd agrolink
```

## 3️⃣ Install Dependencies

```bash
npm install
```

## 4️⃣ Setup Environment Variables

Create a `.env` file and add:

```env
ATLASDB_URL=your_mongodb_url
SECRET=session_secret
CLOUD_NAME=your_cloudinary_name
CLOUD_API_KEY=your_cloudinary_api_key
CLOUD_API_SECRET=your_cloudinary_secret
```

## 5️⃣ Run Project

```bash
node app.js
```

OR

```bash
nodemon app.js
```

# 🎯 Problem Solved

Traditional agriculture systems often include multiple intermediaries which reduce farmers’ profits and increase product prices for consumers.

AgroLink solves this problem by:

* Connecting farmers directly with buyers
* Providing transparent pricing
* Improving farmers’ income
* Making fresh products accessible to consumers
